How it works?

Prerequisites:

SchedulerJS is the most suitable for IT projects executed by teams that consist of 3 types of roles:

  • Product Manager - defining the scope and acceptance criteria for the tasks
  • Software Developer - implementing described tasks
  • Quality Assurance - testing and rating the quality of the work done

Each task should be executed using the following workflow:

Product Manager -> Software Developer -> Quality Assurance

For each task you have to estimate how long does it take to be executed at each stage

What scheduler is doing?

SchedulerJS uses mathematically proven algorithms to compute the order in which the tasks should be executed in order to complete the project in the shortest time
